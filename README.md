# 图像下采样与插值恢复实验
本实验通过对图像进行**下采样**与**插值恢复**，对比分析**有无高斯预滤波**、**不同插值方法**对图像质量的影响，并从**空间域、傅里叶频域、DCT频域**三个维度进行量化与可视化评估。
- 实验人员：2023104747郑煜
## 实验目的
1. 理解图像下采样过程中的**混叠失真**原理
2. 掌握**最近邻、双线性、双三次**三种插值恢复方法
3. 验证**高斯预滤波**对抑制混叠、提升恢复质量的作用
4. 使用 MSE、PSNR 进行图像质量量化评估
5. 通过傅里叶变换与 DCT 观察频域能量变化

## 实验环境
- Python 3
- OpenCV
- NumPy
- Matplotlib

## 实验流程
1. 读取灰度图像
2. 执行两种下采样：
   - 直接下采样（无预滤波）
   - 高斯平滑后下采样
3. 使用三种插值方法恢复原图尺寸：
   - 最近邻内插
   - 双线性内插
   - 双三次内插
4. 计算 MSE、PSNR 评估恢复质量
5. 傅里叶变换 + DCT 频域分析

## 核心算法
### 1. 下采样
- 无滤波直接缩小
- 5×5 高斯滤波预处理 → 缩小（抗混叠）

### 2. 插值恢复
- INTER_NEAREST 最近邻
- INTER_LINEAR 双线性
- INTER_CUBIC 双三次

### 3. 质量评价
- MSE 均方误差
- PSNR 峰值信噪比

### 4. 频域分析
- FFT 傅里叶频谱
- DCT 离散余弦变换
- 低频能量占比统计

## 实验结果
### 1. 量化指标（MSE & PSNR）
- **无预滤波直接缩小**：混叠严重，MSE 高、PSNR 低
- **高斯滤波后缩小**：有效抑制混叠，MSE 显著降低
- 插值效果：**双三次 > 双线性 > 最近邻**

### 2. 空间域对比
- 原图
- 无滤波下采样 / 高斯滤波下采样
- 三种插值恢复结果可视化

### 3. 频域对比
- 原图傅里叶频谱：高频信息丰富
- 下采样后：高频丢失、出现混叠
- 恢复后：频域部分恢复，但无法完全还原

### 4. DCT 低频能量占比
- 原图低频能量占比高
- 下采样+恢复后低频占比略微下降
- 反映图像能量集中在低频，下采样主要损失高频细节

## 实验结论
1. **无预滤波下采样会产生明显混叠失真**
2. **高斯预滤波能有效抑制混叠**，恢复图像质量大幅提升
3. 插值恢复效果：**双三次 > 双线性 > 最近邻**
4. 图像能量主要集中在低频，下采样损失高频细节
5. MSE、PSNR、傅里叶、DCT 可有效评估图像失真程度
